# Youtube dark theme by Vallek

I tried cupple other dark themes and had problems with them so I decided to make another one)

## Compatible with

* [My other Youtube custom style](https://github.com/Vallek/vallek-youtube-custom-css) that hides annoying stuff and let you set number of videos in a row.

## Features:

* Subscriptions screen
* View screen
* Live screen with chat

## Updates

Last update: 11.2025

Tested on stable Chrome and Firefox on Windows 10.
